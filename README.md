# Aircraft Registration Prefixes

Open dataset of **ICAO aircraft registration prefixes** (nationality marks) by country — the letter/number prefix on an aircraft's tail (e.g. `N` = United States, `G` = United Kingdom, `D` = Germany).

**176 country prefixes** in JSON and CSV. Free to use in apps, bots, flight trackers, and aviation projects.

## Files
- [`aircraft-registration-prefixes.json`](aircraft-registration-prefixes.json)
- [`aircraft-registration-prefixes.csv`](aircraft-registration-prefixes.csv)

## Example
```json
{ "prefix": "N", "country": "United States" }
```

## Usage
Map a registration's leading prefix to its country of registry. Useful for plane-spotting tools, flight-tracking apps, METAR/flight-planning utilities, and aviation databases.
